## Week 2 Progress

During the second week of the internship, I started working practically with Power BI. I was introduced to the Power BI interface and learned how to import datasets into the tool, which helped me understand how data is loaded and prepared before starting the analysis process.

I also worked with Power Query, where I practiced several basic data cleaning and transformation techniques. These included tasks such as removing unnecessary rows, trimming and cleaning text values, replacing incorrect or unwanted values, keeping specific rows, removing unnecessary columns, and selecting only the relevant columns needed for analysis. These steps helped me understand how raw data can be transformed into a more structured and organized format.

During this process, I also learned how to identify and handle duplicate records and missing values, which are very common issues when working with real world datasets. By handling these problems, the dataset becomes more reliable and suitable for analysis.

Another important part of this week was learning how to check the dataset for errors and inconsistencies. I also learned about assigning the correct data types to columns, such as text, whole numbers, and decimal numbers. Using the correct data type is important because it ensures that calculations and analysis can be performed accurately.

Through these activities, I gained a better understanding of how data preparation plays a key role in the data analysis process. Before any insights can be generated, the data must first be cleaned, organized, and structured properly.

Overall, the second week helped me become more comfortable working with Power BI and Power Query, and I gained hands on experience in cleaning, transforming, and preparing datasets for further analysis.
